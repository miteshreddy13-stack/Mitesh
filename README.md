# Mitesh
IntelliXO – Mini AI Tic Tac Toe
IntelliXO is a game based on Artificial Intelligence, designed to play the classic Tic Tac Toe using the Minimax algorithm. Users can play against a smart computer opponent with varying difficulty levels, namely Easy, Medium, and Hard. The application features an interactive web-based interface, real-time decision-making, and a scoring system. It demonstrates fundamental concepts of Artificial Intelligence, including game theory and decision-making algorithms, making it a simple yet effective mini AI project.

🚀 Features
🎮 Play against AI with 3 difficulty levels (Easy, Medium, Hard)
🧠 Intelligent AI using Minimax algorithm (Hard mode)
✨ Glassmorphism UI design
🌈 Animated gradient background
🔥 Neon glowing title
📊 Score tracking system
🔄 Restart and Back functionality
🧠 AI & Algorithms Used
Easy Mode
AI selects moves randomly
No decision-making logic
Medium Mode
Rule-based AI

Tries to:

Win if possible
Block opponent's winning move
Hard Mode (Minimax Algorithm)
Uses Minimax algorithm

AI explores all possible game states recursively

Assigns scores:

Win → +1
Draw → 0
Loss → -1
Chooses the move that maximizes its chances of winning

👉 Result: AI plays optimally and becomes unbeatable
